# Onix Snippets

This extension is created for OnixJS framework users, allowing them create Onix application faster.
## Features

This is the list of snippets available in Typescript files:

- onix-microservice
- onix-soa
- onix-boot
- onix-module
- onix-model
- onix-service
- onix-component
